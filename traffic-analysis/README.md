# 📊 Traffic Pattern Analysis

Exploratory data analysis project focused on identifying congestion patterns and mobility trends from real-world traffic datasets.

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn

## Key Focus
- Exploratory Data Analysis (EDA)
- Traffic congestion patterns
- Data visualization
- Trend identification

## Goal
Extract actionable insights from mobility data.
