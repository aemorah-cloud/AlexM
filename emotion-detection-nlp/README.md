# 🧠 Emotion Detection & Sentiment Analysis

NLP project focused on detecting nuanced emotional states using Plutchik’s Wheel of Emotions and Google’s GoEmotions dataset.

## Tech Stack
- Python
- NLP
- Scikit-learn / TensorFlow (if used)
- Pandas

## Key Focus
- Emotion classification beyond positive/negative sentiment
- Affective computing
- Text feature engineering

## Goal
Understand human emotion more deeply through machine learning models.
